# MindAR Library Setup

This folder should contain the MindAR body tracking library.

## Installation

Copy the MindAR library file here:

```bash
mkdir -p public/js
cp node_modules/mind-ar/dist/mindar-body.prod.js public/js/
```

## Required File

- `mindar-body.prod.js` - MindAR body tracking library for AR tattoo try-on functionality

This file is loaded by `/public/ar.html` for the "Try It On" AR experience.
